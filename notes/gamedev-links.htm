Assets

These sites provide partially/completely free assets. Textures, 3d models, sprites, etc.

- kenney nl
Kenney has made a large variety of models, textures and sprites over the years, all of which are free and fall under the CC0 license.
He frequently releases new assets, so checking here often can be worth it.
He accepts donations, has some paid tools/games and also provides a all-in-one package which has all his free assets in one bundle that gets updated with all the new assets when they come out.

- polyhaven

- ambientcg

- the base mesh
A website containing a lot of plain models, all free to download and under the CC0 license.
The models are to scale and have are basically ready to be textured, containing good topology and propper UV unwrapping.

- thoseawesomeguys prompts
Sprites for user inputs. You might have seen these before in a variety of (indy) games, these were designed to be as "generic" as possible.
This, alongside the CC0 license, has made it quite the nicety for using it as a game's input UI.

- Public Domain Image Archive

- texturefabrik

Map making

These sites provide tools and/or resources that have been useful when making levels.

- func_godot
A direct successor to Qodot, func_godot is a tool that allows the importing of quake map files (and source vmf files) into the godot game engine.
It has been quite nice as I find using the hammer editor far easier for level making, especially the early block-out stage.
